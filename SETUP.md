This document explains how to set up the project locally.

## Requirements
- Node.js (v18 or later)
- npm or yarn
- Git

## Installation Steps

1. Clone the repository
2. Install dependencies using `npm install`
3. Copy `.env.example` to `.env`
4. Compile contracts using `npx hardhat compile`
5. Run tests using `npx hardhat test`

## Notes
This project is intended as a starter template for building on the Base blockchain.
